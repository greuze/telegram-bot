# telegram-bot
Telegram simple bot, for messing around with twitter

## Requirements
node.js 10.x.x or above is required

## Config
Set `TELEGRAM_TOKEN` and `TWITTER_BEARER_TOKEN` env vars with proper values, or in a `.env` file.

## Run
```
npm start
```
or
```
node bot.js
```
